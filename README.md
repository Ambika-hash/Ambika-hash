# 🚀 Projects:

### 🗺️ RouteWise
> A location-based safety and routing application that helps users choose safer travel paths in real time. Focuses on risk-aware navigation and using data to improve personal safety through tech.

### 💰 ArthaMind
> A financial awareness and management platform that helps users understand spending, saving, and money habits. Combines logical analysis with user-friendly design to promote smarter financial decisions.

### 📊 Meghalaya Migration & Urban Growth Dashboard
> A real-time analytical tool for government officials that leverages Aadhaar update trends as a proxy for population movement — helping identify infrastructure stress and service gaps before they become critical.

### 🤟 Real-Time Sign Language to Text Converter
> `Raspberry Pi` `Python` `OpenCV`  
> A gesture recognition system for non-verbal communication that converts sign language into text output in real time.

### ♻️ Automated Waste Segregator
> `Arduino` `Sensors`  
> A hardware system that sorts waste into biodegradable and non-biodegradable categories using sensor-based detection and servo-controlled sorting.

### 🔬 ML-Based Predictive Analysis for Resource Utilization
> `Python` `Machine Learning` `Google Cluster Trace v3`  
> Developed an ML pipeline to predict CPU and memory utilization patterns in large-scale computing systems. Research paper authored as part of academic work at Presidency University.
