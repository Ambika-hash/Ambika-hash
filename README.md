## Featured Projects

### RouteWise
A location-based safety and routing application that helps users choose safer travel paths in real time. Focuses on risk-aware navigation and using data to improve personal safety through tech.

### ArthaMind
A financial awareness and management platform that helps users understand spending, saving, and money habits. Combines logical analysis with user-friendly design to promote smarter financial decisions.

A real-time analytical tool for government officials that leverages Aadhaar update trends as a proxy for population movement — helping identify infrastructure stress and service gaps before they become critical.
